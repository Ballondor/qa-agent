# Epic B: 콘텐트 성과 분석 통계

유저 스토리 ID: US-A-8-04
기능: Feature A-8: [FO] 정적 페이지 및 공통 UI
제목: 반응형 UI 및 모바일 최적화
역할: 일반 사용자 (김시민)
목표: 모바일, 태블릿, 데스크톱에서 최적화된 화면으로 서비스를 이용할 수 있다
가치: 어떤 디바이스에서든 편하게 여론조사 데이터를 확인할 수 있다
우선순위: High

인수 조건 (Acceptance Criteria):
  1. Given 모바일 디바이스에서 접속했을 때, When 화면을 확인하면, Then 모바일에 최적화된 레이아웃이 표시된다
  2. Given 모바일에서 그래프를 확인할 때, When 그래프 영역을 터치하면, Then 좌우 스크롤이 가능하다
  3. Given 태블릿에서 접속했을 때, When 화면을 확인하면, Then 태블릿에 적합한 레이아웃이 표시된다
  4. Given 데스크톱에서 접속했을 때, When 화면을 확인하면, Then 데스크톱 전체 화면 레이아웃이 표시된다

유저 스토리 ID: US-B-1-01
기능: Feature B-1: [CMS] 기사/영상 PV 데이터 조회 (QuickSight)
제목: 기사 PV 데이터 조회
역할: 콘텐트 운영자 (이콘텐)
목표: QuickSight 대시보드에서 기사별 PV 데이터를 날짜별/시간별로 조회하고 유입 경로를 확인할 수 있다
가치: 기사별 성과를 분석할 수 있다
우선순위: High

인수 조건 (Acceptance Criteria):
  1. Given QuickSight 대시보드에서, When 기사 PV 영역을 확인하면, Then 기사ID, 제목, 카테고리, PV 수, 유입 경로가 표시된다
  2. Given 기사 목록에서, When 특정 기사를 드릴다운하면, Then 해당 기사의 상세 PV 데이터(날짜별/시간별)가 표시된다
  3. Given 유입 경로 데이터에서, When 경로별 비중을 확인하면, Then JTBC, 네이버, 다음, 네이트, 스포츠빅이벤트, 기타별 PV가 표시된다
  4. Given 일별/주별/월별 필터가 있을 때, When 필터를 변경하면, Then 해당 기간 단위로 집계된 데이터가 표시된다
  5. Given 카테고리 필터가 있을 때, When 특정 카테고리를 선택하면, Then 해당 카테고리의 기사만 표시된다
